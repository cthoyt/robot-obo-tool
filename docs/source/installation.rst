##############
 Installation
##############

The most recent release can be installed from `PyPI
<https://pypi.org/project/robot_obo_tool>`_ with uv:

.. code-block:: console

    $ uv pip install robot_obo_tool

or with pip:

.. code-block:: console

    $ python3 -m pip install robot_obo_tool

*********************
 Installing from git
*********************

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/cthoyt/robot-obo-tool.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/cthoyt/robot-obo-tool.git

****************************
 Installing for development
****************************

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/cthoyt/robot-obo-tool.git
    $ cd robot-obo-tool
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
